# User + JTBD (v0)

## Primary JTBD
When I’m learning AI/ML concepts, help me remember and retrieve them reliably so I can apply them during tasks and assessments.

## Supporting jobs
- Build confidence through quick wins
- See progression clearly (score, mastery, milestones)
- Recover from mistakes without feeling punished

## Top pains to solve
1. Concept overload and poor long-term recall
2. Passive tutorials that don’t verify retention
3. Loss of momentum when sessions are interrupted

## Success moments
- User completes onboarding and can place/recall without confusion
- User replays and improves recall/score in subsequent sessions
- User can explain concept-metaphor mapping after gameplay

## Target personas
1. **Solo learner** — wants fun, practical memory reinforcement
2. **Instructor** — wants measurable concept mastery signals
3. **Builder** — wants conceptual clarity before coding with AI tools
