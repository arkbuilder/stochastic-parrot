# Stochastic Parrot — Product Brief (v0)

## Product name
Dead Reckoning: Memory Sea (repository package name), positioned as Stochastic Parrot program initiative.

## One-line value proposition
A pirate-themed learning game that teaches AI/ML concepts through memory-palace gameplay, turning abstract concepts into repeatable recall under pressure.

## Primary audience (v1)
- Students and self-learners (late high school through early career)
- Educators/tutors who need engaging concept reinforcement
- AI-curious builders who prefer game-based learning to static docs

## Core user problem
AI/ML concepts are hard to retain from passive content alone. Users need an interactive loop that improves recall and confidence, not just exposure.

## Product promise
"Learn by placing and recalling concepts in a playable world, then prove retention through encounters and progression."

## Non-goals (for current phase)
- Not a broad MOOC/LMS replacement
- Not multiplayer-first
- Not certification-heavy at this stage
- Not content breadth beyond current concept curriculum until retention metrics are stable

## Current state (from repo evidence)
- M4 polish + ship readiness complete in docs
- Test, lint, build, and e2e reported passing
- Deployment verification still an external/manual step
