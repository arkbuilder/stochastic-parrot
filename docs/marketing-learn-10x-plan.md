# Marketing Learn Portal — 10x Plan

## Goal
Turn the learning site into a conversion-focused acquisition + education funnel for Stochastic Parrot.

---

## 1) Information Architecture

### Pages
- `/learn` — pillar landing (value proposition + entry points)
- `/learn/start` — fastest path to first success
- `/learn/why` — origin + credibility
- `/learn/how-it-works` — learning loop explanation
- `/learn/curriculum` — concept map + progression
- `/learn/results` — proof + outcomes
- `/learn/faq` — objection handling

### Page-level CTA model
- Every page must end with one primary CTA and one secondary CTA.
- All pages must link to `/learn/start` and `/learn/curriculum`.

---

## 2) Messaging Stack

### Primary message
Learn AI/ML concepts by playing memory challenges instead of passively reading.

### Secondary message
Progression and recall loops make learning measurable and repeatable.

### Trust message
Practical, transparent, and retention-focused — no hype.
