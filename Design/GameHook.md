# **One-Pager**

## **Dead Reckoning: Memory Sea**

### **High Concept**

A legendary pirate-cartographer sails a shattered archipelago called the Memory Sea. Each island is a “locus” in a living memory palace. To master forbidden knowledge of machine intelligence, the player must **place concepts into vivid locations**, then **recall them in moments of danger**—during storms, sea battles, cursed ruins, and giant-squid encounters.

### **Core Fantasy**

You are not a student reading lessons.
You are a pirate captain **charting knowledge into the world** and using memory as your weapon.

### **Why It’s a Banger**

Most “AI learning games” become quizzes with a theme pasted on top.
This one is different:

* **Memory is the core mechanic**
* **Pirates are not cosmetic; they justify navigation, maps, landmarks, and treasure**
* **AI concepts become physical, spatial, and dramatic**
* **It is instantly legible in a 10-second pitch**
* **It fits 8-bit scope extremely well**

### **The Big Twist**

You do not collect knowledge.
You **stage** it.

Example:

* **Training Data** goes in fish crates on the dock
* **Model** goes in the captain’s chart table
* **Inference** goes in the cannon
* **Overfitting** goes in a treasure chest covered in too many barnacles
* **Feedback Loop** goes in a tidewheel or ship’s capstan

Later, the game asks you to survive by remembering where you stored them.

### **Core Player Experience**

1. **Explore** a new island or ship compartment
2. **Encode** 3 AI ideas by placing them in memorable landmarks
3. **Trigger danger**: storm, rival pirate, sea monster, cursed fog
4. **Recall** the correct concept/location under pressure
5. **Resolve the encounter** and earn chart fragments, relics, and ship upgrades
6. **Unlock the next island**, which introduces one new AI idea set

### **Core Gameplay Loop**

```text
SAIL TO NEW ISLAND
        ↓
DISCOVER 3 AI CONCEPTS
        ↓
PLACE THEM IN VIVID LANDMARKS
(memory palace encoding)
        ↓
TRIGGER A THREAT
(storm / battle / squid / ruins)
        ↓
RECALL THE RIGHT LOCATION OR CONCEPT CHAIN
        ↓
SURVIVE + CLAIM TREASURE / MAP FRAGMENT
        ↓
UPGRADE SHIP / OPEN NEW SEA
        ↓
REPEAT WITH SLIGHTLY HARDER CONCEPTS
```

### **What Makes It Feel Good**

The emotional rhythm is strong:

* curiosity while exploring
* delight while placing concepts in absurd pirate landmarks
* tension during danger
* satisfaction when memory becomes mastery

That creates a loop that is both **educational** and **dramatic**.

### **AI Learning Fit**

This works especially well for beginner-friendly AI concepts:

* training data
* model
* inference
* bias
* overfitting
* reinforcement
* feedback loop
* classification

The game teaches by **spatial association**, which is far more memorable than plain exposition.

### **Why AI Doing the Heavy Lifting Helps**

Because AI can generate:

* the islands
* the sprites
* the dialogue
* the encounter variations
* the concept-to-object metaphor mappings
* the micro-content for each lesson

So your competitive edge is not technical difficulty.
It is **design coherence**.

That matters. Judges remember coherence.

### **8-Bit Scope**

This is very buildable:

* top-down overworld sailing
* tiny island maps
* object placement interaction
* lightweight recall encounters
* simple combat/navigation resolution
* strong iconography over heavy simulation

You do **not** need deep physics, pathfinding, or complex combat.
You need:

* memorable places
* readable objects
* pressure moments
* satisfying recall payoffs

### **Best Vertical Slice**

**Island 1: Bay of Learning**

* Concepts: **Training Data**, **Model**, **Inference**
* Landmarks: dock crates, chart table, cannon
* Threat: cursed fog rolls in
* Challenge: remember which object represents which concept to fire the right cannon, align the right chart, and escape the fog
* Boss stinger: first glimpse of the giant squid

This is enough to demo the whole thesis.

### **Onboarding**

Keep the first slice ultra-clean:

* one core interaction at a time
* environmental signposting
* safe first placement
* mild pressure, then recall under stress
* restart in under 5 seconds
* portrait-friendly composition if needed for mobile-first judging 

### **Pitch Line for Judges**

**Dead Reckoning: Memory Sea is an 8-bit pirate adventure that teaches AI by turning the world into a playable memory palace—players don’t answer quizzes, they survive by remembering.**

---

## **Recommendation**

Build the pitch around this sentence:

**“A pirate adventure where memory is navigation, and AI concepts become treasures you must place and recall to survive.”**

That is cleaner than “pirates + learning + Method of Loci.”
It sounds like a real game.

## **3-step Action Plan**

1. Build one island with **3 concepts, 3 landmarks, 1 storm encounter**.
2. Make recall resolve an action scene, not a quiz screen.
3. Make every concept-object pairing visually funny, bold, and unforgettable.

## **Risks / Unknowns**

* If the recall happens in menus, the magic dies.
* If the AI concepts are too abstract, the metaphor layer gets mushy.
* If you add too many systems, you lose the “instant win” pitch.

## **Question you didn’t ask**

What will judges repeat to each other after seeing it?

It should be:
**“Oh, that’s the pirate memory-palace AI game where remembering the world is how you survive.”**

**Pre-mortem:** the biggest failure mode is making it “educational content with pirate art.” The winning version is **an adventure game whose combat/navigation system is memory itself**.
