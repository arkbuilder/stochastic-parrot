# 4-Week Product Plan (starting now)

## Week 1 — Product alignment + baseline instrumentation
- Finalize value prop and audience framing
- Confirm event schema for activation/retention/reliability
- Produce baseline dashboard spec (manual export acceptable)
- Deliverable: Product one-pager + metric definitions signed off

## Week 2 — Onboarding + first-session optimization
- Tighten first 10-minute path (copy clarity, hint timing, friction removal)
- Add explicit "first success" state and next-step CTA
- Deliverable: onboarding improvements + before/after completion comparison

## Week 3 — Retention loop improvements
- Improve Concept Journal surfacing and replay prompts
- Add lightweight "come back" hooks in-session (non-spam)
- Deliverable: D1 replay uplift experiment results

## Week 4 — Ship-readiness review + GTM starter pack
- Validate reliability and guardrails on release candidate
- Prepare educator/learner quickstart guide and demo flow
- Deliverable: release recommendation memo (go / no-go)

## Constraints
- No major feature expansion until Gate A baseline passes
- Scope tradeoff priority: reliability > activation > cosmetics
